# hospital-

Hospital Management System with React frontend, Node.js backend, and MongoDB database.

## Features
- 👨⚕️ Doctors Management
- 🧑🤝🧑 Patients Management
- 📋 Visits Tracking
- 🚑 Ambulances Monitoring
- 🏥 Detection Rooms

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

See DEPLOYMENT.md for setup instructions.
