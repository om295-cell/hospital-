# Hospital Management Frontend

React frontend for the Hospital Management System.

## Setup

1. **Install dependencies:**
   ```
   npm install
   ```

2. **Start development server:**
   ```
   npm start
   ```

3. **Open browser:**
   - App runs at: http://localhost:3000

## Features

- 👨‍⚕️ View Doctors
- 🧑‍🤝‍🧑 View Patients
- 📋 View Visits
- 🚑 View Ambulances
- 🏥 View Rooms
